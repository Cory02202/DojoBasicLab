#WindowsにIBM Cloud CLIをインストール

こちらからWindows 64 を選択インストーラーをダウンロードして実行してください。  
https://github.com/IBM-Cloud/ibm-cloud-cli-release/releases/
(注意：インストーラーが終了するとシステムが再起動されます。作業途中のファイルなどがある場合は予め保存するなどしておいてください）
